SRCF Cloud Desktop Service

Following a few problems with our older NX infrastructure, combined with the unsuitability of modern Linux distributions' use of accelerated 3D graphics for window management, the SRCF has developed a brand new remote desktop service.

The new SRCF Cloud Desktop is powered by the latest secure cloud-based virtualisation technologies: you'll receive sole use of your very own dedicated Xen virtual machine hosted on our in-house private cloud.
