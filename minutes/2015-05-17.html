Alto disk failure

  • Alto had a disk failure in April. It costs about 60 pounds to replace

  • Alto is used to run Cyclone in the Xen pool. However, the disk are only used to boot XenServer and hypothetically contain backups of other VMs, which are currently alive and functioning.
  • The committee sees no need in purchasing a new disk.
  • New disk will not be purchased unless we see a use for it.
  • Worst-case scenario is that XenServer has to be reinstalled.
