SRCF Cloud Desktop Service

Following the success of the new SRCF Cloud Desktop last year, powered by the latest secure cloud-based virtualisation technologies, we have acknowledged a few problems with that trial service and can now present a much more modern system. You'll receive sole use of your very own dedicated cloud server hosted on our in-house private cloud—now with preemptive multitasking!
